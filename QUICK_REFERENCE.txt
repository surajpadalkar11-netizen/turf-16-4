# QUICK REFERENCE - WALLET & PAYOUT SYSTEM

## 🚀 Quick Start Commands

### Start All Services
```bash
# Backend
cd backend && npm start

# Frontend (Users)
cd frontend && npm run dev

# Admin Panel (YOU)
cd admin && npm run dev

# Turf-Admin Panel (Turf Owners)
cd turf-admin && npm run dev
```

## 📊 Database Migrations (Run Once)

In Supabase SQL Editor, run these files in order:
1. `backend/migrations/add_wallet.sql`
2. `backend/migrations/add_admin_settings.sql`
3. `backend/migrations/add_payout_tracking.sql`

## 🔑 Configuration Needed

### 1. Backend .env
```env
GMAIL_USER=your-email@gmail.com
GMAIL_PASS=your-16-char-app-password
```

### 2. Admin Panel Settings (via UI)
- Login to admin panel
- Go to Settings
- Enter Razorpay Key ID
- Enter Razorpay Key Secret
- Save

### 3. Turf Razorpay Accounts
- Each turf needs razorpay_key_id configured
- This is where payouts will be sent

## 💰 Money Flow Summary

```
USER ADDS MONEY:
User → Razorpay Payment → Admin's Razorpay Account → User Wallet Balance

USER BOOKS WITH WALLET:
User Wallet → Booking Payment → Database (payout_status: pending)

ADMIN PROCESSES PAYOUT:
Admin Panel → Transfer → Turf Owner's Razorpay Account → Completed
```

## ⚠️ Important Validations

### Payout Page Shows:
- ✅ Turf name
- ✅ Owner email
- ✅ Owner's Razorpay Key ID (🔑 rzp_live_xxxxx)
- ⚠️ Warning if no Razorpay account
- 🔒 Button disabled if no Razorpay account

### Cannot Process Payout If:
- Turf owner has no Razorpay account
- Payout already completed
- No wallet payment exists
- Admin Razorpay not configured

## 🐛 Troubleshooting

### Razorpay checkout not opening?
- Check browser console for errors
- Verify admin Razorpay keys are saved
- Check browser popup blocker

### Email not sending?
- Verify GMAIL_USER and GMAIL_PASS in .env
- Use Gmail App Password (not regular password)
- Check backend logs

### Payout button disabled?
- Check if turf has razorpay_key_id
- Look for "⚠️ No Razorpay account" warning
- Turf owner must configure their Razorpay account

### "Admin Razorpay not configured" error?
- Go to Admin Panel → Settings
- Enter Razorpay Key ID and Secret
- Save settings

## 📝 Database Fields Reference

### bookings table (payout tracking):
- `wallet_amount_used` - Amount paid via wallet
- `payout_status` - pending/completed/failed
- `payout_amount` - Amount paid to turf owner
- `payout_date` - When payout was processed
- `razorpay_payout_id` - Transfer ID
- `payout_account` - Turf owner's razorpay_key_id

### turfs table:
- `razorpay_key_id` - Turf owner's Razorpay account
- `razorpay_key_secret` - Turf owner's Razorpay secret

### admin_settings table:
- `razorpay_key_id` - Admin's Razorpay Key ID
- `razorpay_key_secret` - Admin's Razorpay Secret
